Aide Page d'alerte La page d'alerte vous permet de voir, très rapidement, toutes les stations qui sont fermées sur le réseau. Page de localisation La carte permet de retrouver toutes les stations du réseau. Vous pouvez sélectionner une station en cliquant sur son point associé ou en la recherchant avec le champs de recherche. Votre position Station ouverte Station fermée Station pleine Station vide Station sélectionnée Lorsqu'une station est sélectionnée, vous pouvez voir les informations de la station et vous pouvez l'ajouter ou la supprimer des favoris. Vous retrouverez alors cette station dans la page "Mes stations". Page des stations favorites Sur cette page vous retrouverez toutes vos stations favorites. Lorsque vous cliquez sur une station, vous pouvez voir toutes les informations associées. Enfin, si vous souhaitez supprimer une station, à partir de la liste, il suffit de faire glisser la ligne vers la gauche. Un bouton de suppression apparait alors.
