Aide Page d'alerte La page d'alerte vous permet de voir, très rapidement, toutes les stations qui sont fermées sur le réseau. Page de localisation La carte permet de retrouver toutes les stations du réseau. Vous pouvez sélectionner une station en cliquant sur son point associé ou en la recherchant avec le champs de recherche en haut à droite "". Votre position Station ouverte Station fermée Station pleine Station vide Station sélectionnée Les stations bonus sont représentées avec un point en leur centre. Rendre son Vélo'v dans une station bonus permet de gagner 30 minutes sur son crédit temps.
Lorsqu'une station est sélectionnée, vous pouvez voir les informations de la station et vous pouvez l'ajouter ou la supprimer des favoris en cliquant sur l'étoile "" ou "". Vous retrouverez alors cette station dans la page "Mes stations". Page des stations favorites Sur cette page vous retrouverez toutes vos stations favorites. Si vous souhaitez supprimer une station, il suffit de faire glisser la ligne vers la gauche. Un bouton de suppression apparait alors. Lorsque vous cliquez sur une station, vous pouvez voir toutes les informations associées. Déclarer un bug Pour reporter un bug, il vous faut une application mail sur votre smartphone avec une adresse configurée. Remplissez les champs ci-dessous puis cliquez sur "envoyer".
